# Welcome to AI Chat System! 🚀

This is your AI-powered chat assistant with integrated document search and research capabilities.

## Features

- 🔍 **Document Search** - Find information across your knowledge base
- 💬 **Natural Conversation** - Chat naturally with AI assistance
- 📚 **Research Mode** - Conduct comprehensive research with analysis
- 🚀 **Unified Mode** - Combine search with intelligent responses

## Chat Modes

Select from different modes using the profile selector:

### 🚀 Unified Search & Chat
Combines document search with conversational AI for the best of both worlds.

### 💬 AI Conversation
Pure conversational AI without document search - great for general questions and discussions.

### 🔍 Document Search
Direct document search with ranked results - perfect when you need specific information.

### 📚 Research Assistant
Comprehensive research mode with deep analysis and citations.

## How to Use

1. **Select a mode** using the chat profile selector
2. **Type your message** in the input box
3. **Get instant responses** powered by RunPod LLM
4. **View search results** when using search-enabled modes

## Powered By

- **Backend:** FastAPI AI Chat Service
- **LLM:** RunPod API
- **Frontend:** Chainlit
- **Deployment:** Railway
